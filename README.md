# DSA Assignment

This repository contains programs for Data Structures and Algorithms assignment.

## Programs Included

### Question 1: Balanced Parenthesis Checker
**File:** `prog_1.c`  
**Description:** Program to check if the given expression has balanced parenthesis using stack.  
**Documentation:** `doc_1.md`

### Question 2: Infix to Postfix Conversion
**File:** `prog_2.c`  
**Description:** Program to convert infix mathematical expression to postfix expression and evaluate it.  
**Documentation:** `doc_2..md`

### Question 3: Reverse Linked List Traversal
**File:** `prog_3.c`  
**Description:** Program to create a linked list and print it in reverse order using recursion.  
**Documentation:** `doc_3.md`

### Question 4: Doubly Linked List Operations
**File:** `prog_4.c`  
**Description:** Program to implement doubly linked list with insertion and deletion operations.  
**Documentation:** `doc_4.md`

### Question 5: Graph Traversal (DFS and BFS)
**File:** `prog_5.c`  
**Description:** Program to implement DFS and BFS traversal on graph using adjacency matrix.  
**Documentation:** `doc_5.md`

### Question 6: Min and Max Heap
**File:** `prog_6.c`  
**Description:** Program to build min heap and max heap from unsorted array.  
**Documentation:** `doc_6.md`

### Question 7: Dijkstra's Algorithm
**File:** `prog_7.c`  
**Description:** Program to implement Dijkstra's algorithm to find shortest path from source node to all other nodes in weighted graph.  
**Documentation:** `doc_7.md`

### Question 8: Sorting Algorithms Comparison
**File:** `prog_8.c`  
**Description:** Program that generates random integers and sorts them using user-chosen algorithm (Bubble Sort, Selection Sort, Insertion Sort, or Merge Sort). Displays comparisons and swaps performed.  
**Documentation:** `doc_8.md`

## How to Compile and Run

To compile any program, use:
```bash
gcc prog_X.c -o prog_X
```

To run the compiled program:
```bash
./prog_X
```

For Windows:
```cmd
gcc prog_X.c -o prog_X.exe
prog_X.exe
```
